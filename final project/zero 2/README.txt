
Pressing c after loading valid file will cycle through different colors. 

Pressing left/right and up/down arrow keys will move light source. 

Program will load appropriately formatted .obj files by entering:
./a0.exe < name.obj

Pressing r will rotate the object in time. Pressing r again will stop the rotation. 

Pressing m will enable mouse control of the zoom and rotation. Pressing m again will 
disable mouse control. 

When mouse control is enabled, clicking right and holding while moving the arrow right 
will zoom in, while moving left is the opposite. Releasing will hold the current zoom. 
Left clicking the mouse will start the object rotating, and releasing will hold the 
current orientation. 

No known bugs.
